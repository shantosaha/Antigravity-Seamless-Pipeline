# Domain Knowledge Setup

Place domain-specific knowledge files in this directory.

## Supported Formats
- Markdown (.md)
- Text (.txt)
- PDF (.pdf)
- JSON (.json)

## Organization

Organize by domain:
```
domain/
├── software_engineering/
│   ├── python_best_practices.md
│   └── api_design.md
├── data_science/
│   ├── ml_algorithms.md
│   └── statistics.md
└── business/
    ├── project_management.md
    └── requirements_gathering.md
```

## Indexing

Knowledge will be automatically indexed for RAG retrieval.
